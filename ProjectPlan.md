Plan for our project

Tamagotchi Style Website

-----Roles:
Dylan Backend
Parker Backend
Jack Frontend
Tess Frontend
Jade Animation and frontend


-----Core Features:
- Virtual pet with mood states based on wellness stats
- Four wellness pillars:
	- Physical
	- Mental
	- Social
	- Spiritual (optional for each user)
- Action/check-in buttons that raise corresponding stats
- Time-based stat decay to create gameplay pressure
- Basic game over + reset loop
- Mobile-friendly responsive layout
- Local storage for saving progress between sessions
- Accessibility support (keyboard navigation + clear labels)




-----Stretch Features:
- User accounts and login
- Daily goals, streak rewards, and achievements
- Pet growth/evolution stages and cosmetics
- Reminder notifications for check-ins
- Social/friend support system (encouragement, shared goals)
- Analytics dashboard of wellness trends over time
- Optional journaling prompts + reflection history
